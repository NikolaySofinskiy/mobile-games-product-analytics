## Бизнес-задачи

### 1. Расчет и автоматизация метрики Retention Rate
Разработана отказоустойчивая функция на Python для расчета удержания пользователей (Retention по дням с момента регистрации). 
* **Входные данные:** Логи регистраций (`reg_data`) и авторизаций (`auth_data`) пользователей.
* **Результат:** Алгоритм, оптимизированный под работу с большими массивами данных, а также визуализация когорт в виде интерактивного или статического Heatmap для быстрого поиска аномалий в удержании.

### 2. Оценка результатов A/B-тестирования акционных предложений
Проанализирован эксперимент по внедрению новых пакетов акций. На основе вводных (рост ARPU в тесте на 5%, но снижение конверсии) осуществлена проверка статистических гипотез.
* **Проведенные шаги:** Оценка нормальности распределения выручки, проверка гомогенности дисперсий, выбор и применение статистических критериев для сравнения средних чеков и долей платящих пользователей.
* **Результат:** Подробный расчет метрик CR, ARPU, ARPPU и аргументированные рекомендации для продуктовой команды по раскатке изменений.

### 3. Разработка фреймворка метрик для внутриигровых событий
Сформирована система продуктовых метрик для оценки результатов ограниченного по времени тематического ивента в игре *Plants & Gardens*:
* Описаны метрики вовлечения, прохождения (funnel analysis) и экономической эффективности события.
* Проведен сравнительный анализ изменения метрик при усложнении механики (введение штрафных откатов по уровням при поражении) и спрогнозировано влияние на отток и монетизацию.
