# mobile-games-product-analytics
Анализ retention, результатов A/B-тестирования игровых акций и определение продуктовых метрик для мобильной игры.


## Авторские права и дисклеймер
Данный проект является финальной аттестационной работой на курсе **«Аналитик данных»** от образовательной платформы **[karpov.courses](https://karpov.courses/)**. 
* Исходные постановки задач, структуры баз данных и тестовые наборы данных принадлежат команде karpov.courses.
* Публикация оригинальных файлов с данными в репозитории ограничена правилами платформы в целях защиты интеллектуальной собственности.
* Код, аналитические выводы, визуализации и методологические решения в данном репозитории выполнены мной самостоятельно.
